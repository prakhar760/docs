openapi: 'PATCH /update_agent/{agent_id}'
---

## API Details

- **URL**: `https://api.openllmhub.com/update_agent/{agent_id}`
- **Method**: `PATCH`

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `agent_id` | string | The ID of the agent to update |

## Request Body

The request body can include the following fields:

| Field | Type | Description |
|-------|------|-------------|
| `agent_name` | string | (Optional) New name for the agent |
| `config` | object | (Optional) Updated configuration for the agent |
| `additional_setting` | object | (Optional) Additional settings for the agent |

### `config` Object Structure

| Field | Type | Description |
|-------|------|-------------|
| `prompt` | string | The updated prompt for the AI |
| `llmmodel` | string | The updated language model to use |
| `voice_id` | string | The updated voice ID |
| `begin_message` | string | The updated initial message |

### `additional_setting` Object Structure

Can include any of the following fields:

- `voice_temperature`
- `voice_speed`
- `responsiveness`
- `interruption_sensitivity`
- `enable_backchannel`
- `backchannel_frequency`
- `backchannel_words`
- `reminder_trigger_ms`
- `reminder_max_count`
- `ambient_sound`
- `ambient_sound_volume`
- `language`
- `boosted_keywords`
- `end_call_after_silence_ms`

## Response

### Success Response

- **Code**: 200
- **Content**: 
```json
{
  "message": "Agent updated successfully"
}
```

### Error Response

- **Code**: 400
- **Content**:
```json
{
  "message": "agent_id is required"
}
```
```json
{
  "error": "no data provided"
}
```

- **Code**: 404
- **Content**:
```json
{
  "message": "No agents found for this agent_id"
}
```

## Example Usage

### cURL
```bash
curl -X PATCH 'https://api.openllmhub.com/update_agent/your-agent-id' \
-H 'Content-Type: application/json' \
-d '{
  "agent_name": "Updated Agent Name",
  "config": {
    "prompt": "New prompt text",
    "llmmodel": "gpt-4",
    "voice_id": "new-voice-id",
    "begin_message": "New greeting message"
  },
  "additional_setting": {
    "voice_speed": 1.2,
    "responsiveness": 0.8
  }
}'
```
### Python
```python
import requests
import json

url = "https://api.openllmhub.com/update_agent/your-agent-id"

payload = json.dumps({
  "agent_name": "Updated Agent Name",
  "config": {
    "prompt": "New prompt text",
    "llmmodel": "gpt-4",
    "voice_id": "new-voice-id",
    "begin_message": "New greeting message"
  },
  "additional_setting": {
    "voice_speed": 1.2,
    "responsiveness": 0.8
  }
})
headers = {
  'Content-Type': 'application/json'
}

response = requests.request("PATCH", url, headers=headers, data=payload)

print(response.text)
```
