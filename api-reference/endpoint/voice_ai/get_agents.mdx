## API Details

- **URL**: `https://api.openllmhub.com/agents`
- **Method**: `GET`
- **`company_id` required**

## Response

### Success Response

- **Code**: 200
- **Content**: 
```json
{
  "agents": [
    {
      "agent_id": "string",
      "agent_name": "string"
    },
    // ... more agents
  ]
}
```

### Error Response
- **Code**: 400
- **Content**: 
```json
{
  "message": "company_id is required"
}
```

- **Code**: 404
- **Content**: 
```json
{
  "message": "No agents found for this company_id"
}
```

## Response fields

| Field | Type | Description |
|--------|------|-------------|
| `agent_id` | string | ID of the Voice AI agent |
| `agent_name` | string | Name of the Voice AI agent |

## Example Usage

### cURL

```bash
curl -X GET 'https://api.openllmhub.com/agents' \
-H "Content-Type: application/json" \
-d '{"company_id": "your company id"}'
```

### Python

```python
import requests

url = "https://api.openllmhub.com/agents"
data = {
    "company_id": "your company id"
}

response = requests.get(url, json=data)

print(response.json())
```
