## API Details

- **URL**: `https://api.openllmhub.com/voice_ai/voice_ai_logs`
- **Method**: `GET`
- **API KEY** `required`

## Response

### Success Response

- **Code**: 200
- **Content**: 
```json
{
  "status": "success",
  "data": [
    {
      "call_id": "string",
      "provider_call_id": "string",
      "agent_id": "string",
      "agent_name": "string",
      "timestamp": "string",
      "transcript": "string",
      "recording_url": "string",
      "call_summary": "string",
      "disconnection_reason": "string",
      "to_number": "string",
      "from_number": "string",
      "call_status": "string"
    },
    // ... more log entries
  ]
}
```

### Response fields

| Field | Type | Description |
|--------|------|-------------|
| `call_id` | string | Unique identifier for the call |
| `provider_call_id` | string | Call ID from the service provider |
| `agent_id` | string | ID of the Voice AI agent |
| `agent_name` | string | Name of the Voice AI agent |
| `timestamp` | string | Start and End Timestamp of the log entry |
| `transcript` | string | Transcript of the call |
| `recording_url` | string | URL to access the call recording |
| `call_summary` | string | Summary of the call |
| `disconnection_reason` | string | Reason for call disconnection |
| `to_number` | string | Recipient's phone number |
| `from_number` | string | Caller's phone number |
| `call_status` | string | Status of the call |

### Error Response

- **Code**: 500
- **Content**:
```json
{
  "status": "error",
  "message": "Error message description"
}
```


## Example Usage

### cURL

```bash
curl -X GET https://api.openllmhub.com/voice_ai/voice_ai_logs \
-H "Authorization: Bearer your_access_token" \
-d 'page=1&limit=10&agent_id=optional_agent_id'
```

### Python 

```python
import requests

url = "https://api.openllmhub.com/voice_ai/voice_ai_logs"

headers = {
    'Authorization': 'Bearer your_access_token'  # Replace with your actual token if required
}

params = {
    'page': 1,
    'limit': 10,
    'agent_id': 'optional_agent_id'  # Replace with a valid agent ID if needed
}

response = requests.get(url, headers=headers, params=params)

print(response.json())
```
