openapi: 'GET /agent/{agent_id}'
---

## API Details

- **URL**: `https://api.openllmhub.com/voice_ai/agent/{agent_id}`
- **Method**: `GET`
- **`agent_id` required**
- **API KEY** `required`

## Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `agent_id` | string | The ID of the agent to retrieve |


## Response

### Success Response

- **Code**: 200
- **Content**: 
```json
{
  "config": object,
  "agent_id": "string",
  "agent_name": "string",
  "additional_setting": object
}
```

### Error Response

- **Code**: 400
- **Content**: 
```json
{
  "message": "agent_id is required"
}
```
- **Code**: 404
- **Content**: 
```json
{
  "message": "Agent not found"
}
```

## Response fields

| Field | Type | Description |
|--------|------|-------------|
| `config` | object | Unique identifier for the call |
| `additional_setting` | object | Additional settings for the agent |
| `agent_id` | string | ID of the Voice AI agent |
| `agent_name` | string | Name of the Voice AI agent |

## Example Usage

### cURL

```bash
curl -X GET https://api.openllmhub.com/voice_ai/agent/{your_agent_id} \
-H "Authorization: Bearer your_access_token"
```
### Python

```python
import requests

# Define the URL for the endpoint
agent_id = 'your_agent_id'  # Replace with a valid agent ID
url = f'https://api.openllmhub.com/voice_ai/agent/{agent_id}'  # Adjust the URL based on your Flask app's running address and port

# Define the headers, including the authorization token if needed
headers = {
    'Authorization': 'Bearer your_access_token'  # Replace with your actual token if required
}

# Send the GET request
response = requests.get(url, headers=headers)

# Print the response status and data
print('Status Code:', response.status_code)
print('Response Data:', response.json())
```
