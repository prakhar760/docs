## API Details

- **URL**: `https://api.openllmhub.com/analysis_routes/get_api_keys`
- **Method**: `GET`
- **Auth Required**: Yes (Bearer token or API key)

## Request Headers

The request must include an `Authorization` header with a valid token or API key.

| Header           | Type   | Required | Description                            |
|------------------|--------|----------|----------------------------------------|
| `Authorization`  | string | Yes      | Bearer token or API key for authorization |

## Responses

### Success Response

- **Code**: 200
- **Content**:
```json
{
  "status": "success",
  "api_keys": [
    {
      "api_key_id": "string",
      "api_key": "string",
      "status": "string",
      "created_at": "string"
    }
  ]
}


### Error Response


- **Code**: 403
- **Content**: 
```json
{
  "status": "failure",
  "message": "Invalid authorization type"
}


```
- **Code**: 404
- **Content**:
```json
{
  "status": "failure",
  "message": "Company not found"
}



```

- **Code**: 500
- **Content**:
```json
{
  "status": "failure",
  "message": "Server error: Error message description"
}



```

### cURL

```bash
curl -X GET 'https://api.openllmhub.com/analysis_routes/get_api_keys' \
-H 'Authorization: Bearer your-token' \
-H 'Content-Type: application/json'


```

### Python 

```python
import requests

url = "https://api.openllmhub.com/analysis_routes/get_api_keys"
headers = {
  'Authorization': 'Bearer your-token',
  'Content-Type': 'application/json'
}

response = requests.request("GET", url, headers=headers)

print(response.text)


```
